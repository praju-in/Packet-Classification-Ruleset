# Packet-Classification-Ruleset
Creation of Dataset (Packet Ruleset) Packet classification it is method of segregation of incoming packet, for that we need Ruleset. Base on that Ruleset this classification is perform. I am also working on classification project, where I need Ruleset not available publicly. so I decided to create my own Dataset.
